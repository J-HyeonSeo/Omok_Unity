# Omok Client With Unity Engine

```
유니티 엔진을 사용하여 오목게임의 클라이언트를 구현한 프로젝트 입니다.
유니티 엔진의 UI구성방법, 기본적인 활용방법을 정리하기 위해서 만들어졌습니다.
자세한 내용은 블로그를 통해 확인 부탁드립니다.
```
